FAQ

What is this?

LibSearch is a search engine that searches only in a custom scope. It helps user find the result they want, presented in a clean and simple way without going through repetitive copy-paste-search process.

How to use?

  1. Add the URL of the website you want to search in the search engine.
  2. Enter the keyword you want to search in the search box.
  3. Click on the search button and get your search results.
